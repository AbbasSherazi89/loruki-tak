# Loruki Cloud Hosting

A modern, responsive landing page for "Loruki" - a fictional cloud hosting platform. This project showcases a clean, feature-rich web interface built with semantic HTML5 and custom CSS.

Live Demo: https://luminous-cranachan-9f5b51.netlify.app/

## ✨ Features

- **Fully Responsive Design:** Adapts seamlessly to desktop, tablet, and mobile devices.
- **Modern UI/UX:** Clean layout with a focus on typography, spacing, and visual hierarchy.
- **Feature Showcase:** Dedicated section to highlight key platform features using icons and descriptive text.
- **Reusable Component Structure:** Organized with utility classes and component-based styling (navbar, cards, footer).
- **Font Awesome Integration:** Retina-ready vector icons from Font Awesome 6.5.1.
- **Cross-Page Navigation:** Structured with links for Home, Features, and Docs pages.

## 🛠️ Technologies Used

- **HTML5** - Semantic markup for structure.
- **CSS3** - Custom styling with Flexbox & Grid layouts.
- **Font Awesome 6.5.1** - Icon library for visual enhancements.
- **Google Fonts** (implied via `utilities.css`) - Modern typography.
